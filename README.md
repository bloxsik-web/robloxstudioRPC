# Roblox Studio RPC

Discord Rich Presence для Roblox Studio.

Показывает:

- `Editing "PLACE_NAME"`
- `Idle`

## Использует

- Python
- pypresence
- pygetwindow

## Использование

Скачать `RobloxRPC.exe` из релизов и запустить.

Программа автоматически ждёт открытия Roblox Studio и обновляет статус в Discord.

## Примечание

Discord должен быть открыт.
